El navegador está mostrando la versión ANTIGUA de app.js
debido al caché.

OPCIONES PARA FORZAR RECARGA:

1. CTRL + SHIFT + R (Windows/Linux)
2. CTRL + F5 (Windows)
3. CMD + SHIFT + R (Mac)

O MANUALMENTE:

1. Abre DevTools (F12)
2. Ve a la pestaña "Network"
3. Marca el checkbox "Disable cache"
4. Recarga la página (F5)
